# 隱私權政策 (Privacy Policy)

**最後更新日期：2025-12-10**

感謝您使用「自動填寫員工編號」Chrome 擴充功能(以下簡稱「本擴充功能」)。我們非常重視您的隱私。本隱私權政策旨在說明我們如何處理您的資訊。

## 1. 資訊的收集與使用

本擴充功能的核心目的是為了簡化您的日常操作。為了實現此目的:

-   **我們收集的資訊**:本擴充功能會要求您提供一項資訊:您的「員工編號」。
-   **資訊的用途**:您所提供的「員工編號」僅用於在您瀏覽 `check.springhouse.com.tw` 網站時,自動為您填寫對應的欄位。

除此以外,我們不會收集、追蹤或儲存任何其他的個人身份資訊、瀏覽紀錄或使用數據。

## 2. 資訊的儲存與安全

**您的資料安全是我們的首要考量。**

-   **本地儲存**:您所輸入的「員工編號」將使用 Chrome 提供的 `chrome.storage` API **完全儲存在您的本機電腦上**。
-   **絕不傳輸**:這意味著您的員工編號**永遠不會**被傳輸、發送或揭露給我們的伺服器或任何第三方。所有資料處理均在您的瀏覽器內部完成。
-   **安全性**:由於資料僅儲存在您的裝置上,其安全性取決於您個人電腦和瀏覽器的安全措施。

## 3. 權限要求說明

本擴充功能在安裝時會向您請求以下權限:

-   `storage`:此權限是必要的,以便將您的員工編號安全地保存在您的瀏覽器儲存空間中,避免您每次都需要重新輸入。
-   `content_scripts` on `*://check.springhouse.com.tw/*`:此權限允許本擴充功能在指定的網站上運行其核心腳本,以實現自動尋找輸入框、填寫和提交的功能。本擴充功能不會在任何其他網站上運行。

## 4. 第三方服務

本擴充功能不使用任何第三方服務、分析工具或廣告網路。

## 5. 隱私權政策的變更

我們可能會不時更新本隱私權政策。任何變更都將發布在此頁面上,我們建議您定期查看。
